# serverEncryptFiles
**Version:** 1.0.0

## Description
Is a app for servers for encrypt files in userFiles folder, created with this app.
This app is designated for local enviroments.

## Requirements
- python installe in your system.
- cryptography python library.
- streammlit python library.

## Download
(https://github.com/RedDarkdevXD/serverEncryptFiles.git)

## Usage
# Linux
Run setup.sh and follow the instructions.

# Windows
1. Install python.
2. Install librarys requires for the app.
    - cryptography python library.
    - streammlit python library.
3. Create folder 'userFiles' and 'credentialsFiles' in the app directory.
4. Create cred.txt in 'credentialsFiles' directory. In that file put the password what you want for acces app.
5. Run this command "streamlit run streamlit_app.py" in the app directory.
6. Generate a key with the app interfaces.
